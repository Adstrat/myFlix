myFlix App

An application which displays movies, directors and genres.

Users can sign up, update personal information, and create a list of their favourite movies.

REQUEST URL HTTP Method Request Body Data Format Responce Body Data Format
Return a list of ALL films /films GET NONE A JSON object about all the films. Example:
{
title: "Bladerunner",
description: "Set in a dystopian future, a cop hunts down fugitive, synthetic humans.",
genre: "80s scifi",
director: "Ridley Scott",
image:
featured: }
Return information about ONE film /films/:title GET NONE A JSON object about the film. Example:
{
"title": "Bladerunner",
"description": "Set in a dystopian future, a cop hunts down fugitive, synthetic humans.",
"genre": "80s scifi",
"director": "Ridley Scott",
"image":
featured: 1 }
Return genre from title of film /films/genres/:title GET NONE {
"title": "Aliens",
"Genre": "SciFi" }
Return information about a director /films/:director GET NONE "director": "Mike Hodges",
"bio": "Mike Hodges (born 29 July 1932) is an English screenwriter, film director, playwright and novelist."
Allow users to register /users POST JSON { "username": "AStrat",
"name": "Andrew Stratton",
"email": "youremail@gmail.com" }
Allow users to update their information /users/:username POST JSON { "username": "AStrat",
"name": "Andrew Stratton",
"email": "youremail@gmail.com" }
Allow users to ADD a film to their favourites /users/:username/:film PUT NONE Sucessful PUT request adding film 'FILM NAME' to user 'USERNAME'
Allow users to REMOVE a film from their favourites /users/:username/:film DELETE NONE Sucessful REMOVE request deleting film 'FILM NAME' from user 'USERNAME's favourite list
Allow users to deregister /users/:username DELETE NONE Account deleted for 'USERNAME'
